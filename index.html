Braille CSS

Braille CSS is a specification for expressing how to format braille, based on CSS 2 and 3. The specification can be viewed at https://braillespecs.github.io/braille-css.

OBFL

OBFL is a specification for expressing how to format braille, inspired by XSL-FO. The specification can be viewed at https://braillespecs.github.io/obfl.

PEF

PEF is a specification for expressing braille books accurately. The specification can be viewed at https://braillespecs.github.io/pef.

See also

For applications based on these specifications, see https://brailleapps.github.io
