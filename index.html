Provides specifications for producing braille

Braille CSS

Braille CSS is a specification for expressing how to format braille, based on CSS 2 and 3. The specification can be viewed at http://braillespecs.github.io/braille-css.

OBFL

OBFL is a specification for expressing how to format braille, inspired by XSL-FO. The specification can be viewed at http://braillespecs.github.io/obfl.

PEF

PEF is a specification for expressing braille books accurately. The specification can be viewed at http://braillespecs.github.io/pef.
